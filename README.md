# html2csv

Transform HTML to CSV.

## Usage

```
Usage: html2csv [OPTIONS] FILE
  -d, --delimiter string   delimiter (default ",")
```

## Notes

- If a file is not specified, read from stdin
- The delimiter must be a single character
